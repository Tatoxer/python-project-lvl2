## Usage
* gendiff [-h] [-n] [-f] file1 file2

### positional arguments:
  * **file1** - path/to/file1
  * **file2** - path/to/file2

### optional arguments:
  * **-h, --help**:            show this help message and exit
  * **-n, --no_colors**:       Print non colored result
  * **-f , --format**: set printing format resultavailable: "plain", "dictionary", "json". Default: plain
